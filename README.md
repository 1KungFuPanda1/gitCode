## 🍅 番茄钟

一个可直接在 Windows 桌面使用的番茄工作法计时器。

### 功能

- ⏳ **倒计时**：设定专注时间，到时自动提醒
- ⏱ **正计时**：从零开始累计，自由记录工作时长
- 🎵 **声音提醒**：倒计时结束自动播放悦耳提示音
- ⏭️ **快进**：可手动跳过当前计时
- ⏸️ **暂停/重置**：灵活控制计时节奏
- ⏱ **预设时间**：5 / 15 / 25 / 45 / 60 分钟一键切换
- ✏️ **自定义时间**：支持 1–120 分钟任意设定

### 使用方式

1. 下载 `番茄钟.html` 到桌面
2. 双击用浏览器打开
3. 无需安装任何依赖

### 技术栈

纯 HTML/CSS/JS，单文件零依赖，Web Audio API 实现音效。
